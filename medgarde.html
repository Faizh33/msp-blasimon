Pour joindre le médecin de garde

La garde du week-end débute le samedi à partir de midi.
La nuit, en semaine, ou le week end, le médecin de garde est appelé uniquement sur appel du centre 15.
Vous trouverez ci dessous le nom des médecins de garde pour les dimanches et jours fériés sous réserve de changement.
Dans ce cas vous pouvez appeler au cabinet médical 05 56 71 52 13, le répondeur vous donnera le nom et le numéro de téléphone de ce médecin. Vous pouvez vous rendre sur le site du conseil de l'ordre ou bien aussi appeler le 15.
